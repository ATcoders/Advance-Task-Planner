# Advanced Task Planner

A feature-rich desktop task management application designed to help users build consistency, track daily goals, and analyze their productivity. Built with Python Tkinter, the application provides an interactive task dashboard with visual progress tracking and offline data storage.

## 🚀 Features

- ✅ Create and manage daily tasks
- 📊 Monthly and weekly progress bars to keep a track of consistency 
- ☑️ Interactive daily completion tracking using checkboxes
- 💾 Local SQLite database for secure offline data storage
- 🏆 Automatic achievement certificate generation for maintaining 90%+ consistency
- 🖥️ Desktop application with executable (.exe) support

## 🛠️ Technologies Used

- **Python** – Core application logic
- **Tkinter** – Graphical User Interface
- **SQLite** – Local database management
- **Matplotlib** – Progress visualization and analytics
